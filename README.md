# Contributing

## Installation

```powershell
# Clone repository into ./trustworthytimes folder
git clone https://github.com/darccyy/trustworthytimes.git

# Install dependencies
npm run install
```

## Development

```powershell
# Start development server
npm run start:dev
```

Save file to update server

## Production

```powershell
# Build production
npm run build

# Start production server
npm run start
```
